# 芒格检查清单

芒格是检查清单的坚定拥护者："飞行员有检查清单，外科医生有检查清单。如果这些聪明、训练有素的人都需要检查清单，你凭什么认为你不需要？"

## 一、投资前检查清单

### 业务质量
- [ ] 我能用一两句话解释清楚这个业务吗？
- [ ] 企业有可理解的、持久的竞争优势吗？
- [ ] 护城河在变宽还是变窄？
- [ ] 企业产生正的自由现金流吗？
- [ ] 企业的增长是否可持续（非一次性因素）？
- [ ] 这个行业的结构是否有利于长期盈利？

### 管理层
- [ ] 管理层有良好的长期业绩记录吗？
- [ ] 管理层诚实且对股东坦诚吗？
- [ ] 管理层的利益与股东一致吗？（看持股、薪酬结构）
- [ ] 管理层在资本配置上有良好记录吗？
- [ ] 管理层是否保持合理的自信而非自满？

### 财务健康
- [ ] 资产负债表是否健康？（合理的债务水平）
- [ ] 现金流是否持续且可预测？
- [ ] 投资回报率（ROIC）是否持续高于资本成本？
- [ ] 利润率是否稳定或在改善？
- [ ] 是否存在隐藏的表外负债或风险？

### 估值
- [ ] 当前价格相对于内在价值有多大安全边际？
- [ ] 与历史估值水平相比如何？
- [ ] 与可比公司相比估值如何？
- [ ] 即使我的估值假设错了20%，这仍然是好的投资吗？

### 风险（逆向思维）
- [ ] 什么因素可能摧毁这个投资的价值？
- [ ] 如果最坏情况发生，我能承受多大损失？
- [ ] 这个投资对经济周期有多敏感？
- [ ] 有哪些我可能没有想到的风险？
- [ ] 我是否被某种心理偏误所影响？

## 二、心理偏误自查清单

在做重大决策前，逐条检查：

### 信息获取阶段
- [ ] 我是否在选择性地收集信息？（确认偏差）
- [ ] 我是否被容易想到的信息过度影响？（可得性偏差）
- [ ] 我是否被第一个数字/信息锚定了？（锚定效应）
- [ ] 我的信息来源是否足够多元？

### 分析判断阶段
- [ ] 我是否过度自信？（我真的比市场更了解这件事吗？）
- [ ] 我是否在用单一工具分析问题？（锤子-钉子倾向）
- [ ] 我是否把复杂问题过度简化了？
- [ ] 我能否比反对者更好地阐述他们的论点？

### 决策执行阶段
- [ ] 我是否因为"大家都在做"而跟风？（从众效应）
- [ ] 我是否因为害怕损失而犹豫不决？（损失厌恶）
- [ ] 我是否因为已经投入了很多而不愿放弃？（沉没成本谬误）
- [ ] 我是否因为害怕错过（FOMO）而匆忙决定？
- [ ] 我是否因为竞争心态而出价过高？
- [ ] 是否有多种偏误在同时影响我？（Lollapalooza效应）

### 回顾反思阶段
- [ ] 我是否在事后说"我早就知道"？（后视偏差）
- [ ] 我是否诚实地记录了决策过程和理由？
- [ ] 我是否从错误中学到了真正的教训？

## 三、商业决策检查清单

### 进入新市场/新业务
- [ ] 这个市场/业务在我的能力圈内吗？
- [ ] 竞争格局如何？是否已经过度拥挤？
- [ ] 我们的差异化优势是什么？
- [ ] 进入成本和退出成本分别是多少？
- [ ] 这个决策对核心业务有何影响？
- [ ] 如果失败，最坏的结果是什么？能承受吗？

### 避免的商业错误
- [ ] 避免进入竞争激烈且无差异化的业务
- [ ] 避免依赖单一关键人物的业务
- [ ] 避免快速变化且无法预测的行业
- [ ] 避免需要持续高投入才能维持的业务
- [ ] 避免政策高度不确定的行业
- [ ] 避免产品容易被复制的业务
- [ ] 避免客户转换成本低的业务

## 四、人生重大决策检查清单

### 职业选择
- [ ] 这个方向是否在我的能力和兴趣范围内？
- [ ] 这个行业的长期前景如何？（避免衰退行业）
- [ ] 竞争程度如何？（避免过度拥挤的赛道）
- [ ] 这份工作能让我持续学习和成长吗？
- [ ] 10年后我会为这个选择感到骄傲还是后悔？
- [ ] 我是否被薪资锚定而忽视了其他重要因素？

### 重大财务决策
- [ ] 这个决策是否超出了我的能力圈？
- [ ] 最坏的情况是什么？我能承受吗？
- [ ] 我是否考虑了机会成本？
- [ ] 我是否被他人的行为所影响？（从众）
- [ ] 我是否给自己留了安全边际？
- [ ] 这是可逆的还是不可逆的决策？

### 合作伙伴/伴侣选择
- [ ] 这个人诚实正直吗？（芒格认为这是最重要的）
- [ ] 我们的价值观是否一致？
- [ ] 我们的能力是否互补？
- [ ] 我是否忽视了明显的危险信号？
- [ ] 我是否因为沉没成本而不愿离开不好的关系？

## 五、决策后复盘清单

每个重要决策做完后，定期回来检查：

- [ ] 当初的决策理由是否仍然成立？
- [ ] 实际结果与预期有何偏差？为什么？
- [ ] 我犯了哪些思维错误？
- [ ] 环境是否发生了变化，需要调整？
- [ ] 我从这次决策中学到了什么？
- [ ] 下次遇到类似决策，我会怎么做？

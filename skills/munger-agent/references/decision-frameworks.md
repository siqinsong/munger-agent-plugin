# 芒格决策框架

## 一、企业分析五步法（来自可口可乐案例分析）

芒格在分析可口可乐时展示了他的通用企业分析框架，这个框架适用于分析任何企业：

### 第一步：理解业务本质
- 这是什么生意？用最简单的话描述
- 商业模式如何运转？钱从哪里来？
- 价值链中的关键环节是什么？

### 第二步：分析竞争优势（护城河）
- 品牌优势：消费者对品牌的认知和忠诚度
- 规模优势：规模带来的成本优势是否显著
- 网络效应：使用者增加是否增强产品价值
- 转换成本：客户切换到竞争对手的代价
- 知识产权：专利、配方、技术壁垒
- **关键问题**：护城河是在变宽还是变窄？

### 第三步：评估管理层
- 能力：管理层的业务能力和决策记录
- 诚信：是否坦诚对待股东，有无自利行为
- 激励对齐：管理层的利益是否与股东一致
- 资本配置：管理层在资本配置上的能力如何

### 第四步：财务分析
- 盈利能力：ROE、ROA、利润率的趋势
- 现金流：自由现金流是否充沛且持续
- 资产负债表：债务水平是否健康
- 增长质量：增长是来自内生还是并购

### 第五步：估值与安全边际
- 内在价值估计：基于保守的现金流预测
- 安全边际：当前价格相对于内在价值的折扣
- 与机会成本比较：是否是目前最好的投资机会

## 二、投资决策通用框架

### 进入决策
1. **这在我的能力圈内吗？**——如果不是，停止。
2. **业务本身好不好？**——有没有持久的竞争优势？
3. **管理层好不好？**——能力、诚信、对齐。
4. **价格够便宜吗？**——有没有足够的安全边际？
5. **逆向检查**——什么会让这个投资变得一文不值？

### 持有/卖出决策
- "如果我现在没有持有，我会以当前价格买入吗？"
- 投资论点是否已经改变？
- 是否出现了更好的机会（机会成本）？
- 是否因为心理偏误（损失厌恶、禀赋效应）而不愿卖出？

### 仓位大小决策
芒格的集中投资哲学：
- 最好的想法值得更大的仓位
- 真正理解的投资可以重仓
- 不理解的投资应该放弃而非小仓位尝试
- "多元化是无知的保护。如果你知道自己在做什么，多元化就没有意义。"

## 三、商业决策框架

### 市场进入决策
1. 这个市场的吸引力如何？（大小、增长、利润率）
2. 竞争格局如何？（集中度、进入壁垒）
3. 我们的竞争优势是什么？（与现有玩家相比）
4. 进入成本是多少？退出成本呢？
5. 最坏的情况是什么？能承受吗？

### 定价决策
- 理解成本结构和竞争态势
- 评估消费者感知价值（他们愿意付多少）
- 长期市场地位 vs 短期利润的权衡
- 避免价格战——"价格战伤害所有参与者"

### 人才/合作伙伴选择
芒格的标准：
1. **诚信第一**——"如果你雇了一个聪明但不诚实的人，你最好希望他又懒又笨"
2. **能力匹配**——能否胜任工作
3. **价值观对齐**——长期合作的基础
4. **互补性**——各有所长

## 四、人生重大决策框架

### 可逆性评估法
将决策分为两类：
- **可逆决策**：可以回头，应该快速决策、大胆尝试
- **不可逆决策**：不能回头，应该慎重思考、充分分析

### 10/10/10法则（与芒格的长期思维一致）
做决策时问自己：
- 10分钟后我会怎么想？
- 10个月后我会怎么想？
- 10年后我会怎么想？

### 机会成本分析
- 选择A意味着放弃什么？
- 被放弃的最佳选项是什么？
- 如何最小化机会成本？

### 芒格的人生决策原则
1. **在有鱼的地方钓鱼**：选择好的赛道比在差的赛道上努力更重要
2. **复利人生**：好的习惯、关系、知识会复利增长
3. **避免毒药**：远离毒品、酒精依赖、赌博、有毒的人
4. **配得上你想要的**："得到你想要的东西的最可靠方法，是让自己配得上它"
5. **持续学习**："每天睡觉前，都要比醒来时聪明一点"

## 五、逆向思维实操框架

### 步骤
1. **列出失败因素**：穷举"如果要在这件事上彻底失败，我会怎么做？"
2. **评估概率**：每个失败因素发生的概率有多大？
3. **评估影响**：每个失败因素的破坏力有多大？
4. **设计防御**：如何预防概率高且破坏力大的失败因素？
5. **设置预警**：设置什么信号可以提前发现问题？

### 经典应用
- 投资中："这家公司可能因为什么而衰败？"
- 创业中："这个项目最可能在哪里失败？"
- 职业中："什么会让这份工作变成噩梦？"
- 人生中："什么行为会毁掉我的人生？——那就避免它们。"
